# Scala Steward

Project using [GitHub Actions](https://github.com/features/actions) to run [Scala Steward](https://github.com/scala-steward-org/scala-steward) through the [Scala Steward GitHub Action](https://github.com/scala-steward-org/scala-steward-action).<br>
The workflow runs according to [schedule](.github/workflows/scala-steward.yml#L4) for [repositories](https://github.com/organizations/maginepro/settings/installations) to which the application has access.
